# Rust - Módulo Básico

## Objetivo

Este módulo apresenta os fundamentos de Rust, desde a instalação até conceitos essenciais como variáveis, tipos de dados, funções e controle de fluxo.

---

## Estrutura do Módulo

### 1. Introdução
**Arquivo:** `introducao/README.md`

Conceitos essenciais:
- O que é Rust e por que aprender
- Características principais (ownership, borrow checker, type system)
- Quando usar Rust
- Breve história

### 2. Aula 01 - Primeiros Passos
**Arquivo:** `aula01/README.md`

Tópicos:
- Instalação e setup
- Cargo e estrutura de projeto
- Primeiro programa (Hello, World!)
- Variáveis e tipos primitivos
- Println! macro
- Operações matemáticas

### 3. Aula 02 - Variáveis e Mutabilidade
**Arquivo:** `aula02/README.md`

Tópicos:
- Declaração de variáveis
- Imutabilidade (padrão)
- Mutabilidade explícita
- Sombreamento de variáveis
- Constantes

### 4. Aula 03 - Tipos de Dados
**Arquivo:** `aula03/README.md`

Tópicos:
- Inteiros com e sem sinal
- Ponto flutuante
- Booleanos
- Caracteres
- Tuplas
- Arrays

---

## Fluxo de Aprendizado

1. Comece com **Introdução** para entender o contexto
2. Siga para **Aula 01** para setup e primeiro programa
3. Complete as **Aulas subsequentes** em sequência
4. Faça os exercícios propostos em cada aula
5. Pratique com pequenos projetos

---

## Pré-requisitos

- Linux, macOS ou Windows
- Terminal/Command Prompt
- Editor de texto ou IDE

---

## Como usar este módulo

Cada aula contém:
- Conceitos explicados
- Exemplos de código executável
- Exercícios práticos
- Links para recursos adicionais

Execute os exemplos:

```bash
cd Rust/basico/aula01
rustc exemplo.rs
./exemplo  # Linux/macOS
# ou
exemplo.exe  # Windows
```
