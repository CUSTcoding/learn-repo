# learn-repo

Este repositório reúne meus projetos de estudo e experimentação em desenvolvimento de software, ciência de dados e tecnologias fullstack. Ele está organizado por linguagem e área de conhecimento, permitindo acompanhar o progresso de aprendizado, testar novas bibliotecas e frameworks, e manter exemplos práticos de cada tecnologia.

## Estrutura geral

**Linguagens:** Java, Python, Javascript, Rust, Typescript e Shell
Exercícios básicos, algoritmos, estruturas de dados e orientação a objetos.

**Frameworks e bibliotecas:**
Spring Boot, Axum, Rust Bitcoin, Django, PyTorch, Next.js, React Native, entre outros.

**Fullstack e Mobile:**
Projetos completos de front-end e back-end, animações, sites 3D e integração com APIs.

**Data Science:**
Machine Learning e Deep Learning, com scripts, notebooks e exemplos práticos.

**Scripts e automações:**
Pequenos utilitários e experiências em Python e Shell.

**Shell / Administração de Sistemas:**
Scripts e exemplos de automação, comandos Linux/Windows, administração de sistemas e ferramentas para suporte técnico.
