# Python - Módulo Básico

## Objetivo

Este módulo apresenta os fundamentos de Python, desde a instalação até conceitos essenciais como variáveis, tipos de dados, funções e controle de fluxo. O objetivo é fornecer uma base prática para aplicações de backend, automação e ciência de dados.

---

## Estrutura do Módulo

### 1. Introdução
**Arquivo:** `introducao/README.md`

Conceitos essenciais:
- O que é Python e onde usá-lo
- Características principais (sintaxe, dinamismo, bibliotecas)
- Quando escolher Python

### 2. Aula 01 - Primeiros Passos
**Arquivo:** `aula01/README.md`

Tópicos:
- Instalação e setup
- Criação de ambientes virtuais
- Primeiro script (Hello, World!)
- Execução de arquivos Python
- Variáveis e tipos básicos

### 3. Aula 02 - Variáveis e Mutabilidade
**Arquivo:** `aula02/README.md`

Tópicos:
- Declaração de variáveis
- Tipos mutáveis vs imutáveis
- Convenções de nomes
- Escopo e lifetime de variáveis

### 4. Aula 03 - Tipos de Dados
**Arquivo:** `aula03/README.md`

Tópicos:
- Números, strings
- Listas, tuplas, conjuntos e dicionários
- Operações comuns e métodos úteis

### 5. Aula 04 - Funções e Módulos
**Arquivo:** `aula04/README.md`

Tópicos:
- Definição de funções, argumentos e retorno
- Funções anônimas (lambda)
- Módulos e pacotes
- Importações e boas práticas

### 6. Aula 05 - Condições
**Arquivo:** `aula05/README.md`

Tópicos:
- Estruturas condicionais (`if`/`elif`/`else`)
- Operadores relacionais e lógicos
- Associação (`in`) e identidade (`is`)

### 7. Aula 06 - Loops
**Arquivo:** `aula06/README.md`

Tópicos:
- `for` e `while`
- `range()`, `enumerate()`, `zip()`
- Controle de loop (`break`, `continue`)
- Compreensões e generators

---

## Fluxo de Aprendizado

1. Leia `introducao/README.md` para entender o contexto
2. Siga para `aula01` para setup e primeiro script
3. Progrida nas aulas em sequência e faça os exercícios
4. Pratique com pequenos projetos e scripts de automação

---

## Como executar exemplos

Criar e ativar ambiente virtual (recommended):

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate    # Windows
pip install -r requirements.txt  # se houver
```

Executar um script:

```bash
python exemplo.py
```
