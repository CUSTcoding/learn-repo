# Introdução ao Python - Primeiros Passos

## O que é Python?

Python é uma linguagem de alto nível, interpretada e de tipagem dinâmica, muito usada em automação, back-end, ciência de dados e scripts.

## Por que aprender Python?

- Sintaxe clara e legível, ótima para iniciantes
- Amplo ecossistema de bibliotecas (Pandas, NumPy, Flask, Django, PyTorch)
- Excelente para prototipagem rápida e automação
- Grande comunidade e muitos recursos de aprendizado

## Pré-requisitos

- Python 3.8+ instalado
- Editor de código (VS Code, PyCharm, Vim)
- Terminal/Command Prompt
